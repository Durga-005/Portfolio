  • Image Classification using CNN
    Image Classification is the task of assigning a label or category to an input image. For example, given an image of a cat or dog, the model predicts whether it’s a “cat” or “dog.” Convolutional Neural Networks (CNNs) are a type of deep learning model specifically designed for analyzing visual data. CNNs automatically learn spatial hierarchies of features from images, which makes them very effective for image classification tasks.
  • Moving Object Dedection
    Moving Object Detection (MOD) is a computer vision task that involves identifying and tracking objects that are moving in a video or a sequence of images. It is widely used in surveillance systems, traffic monitoring, autonomous vehicles, and robotics. The goal is to detect regions of interest (moving objects) in real-time while ignoring the static background.
